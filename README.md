#### Article-Crawler 🕷️

This is a scrapy project which crawls `http://tutsplus.com/` and spits out an csv file with the fields - author, published_date and title

###### Dependency:

Scrapy 

```
pip install scrapy
```

###### Running the crawler:

``` 
scrapy crawl tutspider -o out.csv 
```
