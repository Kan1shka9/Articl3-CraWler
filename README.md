## Articl3-CraWler
This is a scrapy project which crawls <a><i>http://tutsplus.com/</i></a> and spits out an csv file with the fields - author, published_date, title

###### Dependency:
Scrapy ``` pip install scrapy ```

###### Running the crawler:
``` scrapy crawl tutspider -o out.csv ```
